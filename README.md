"# SupplyHouseFramework" 

Requirements:
jdk 11
Cucumber-java 7.5.0
junit 4.11
testng 6.14.3
Selenium java 4.8.0
cucumber-testng 7.5.0
Apache Maven 3.8.6

How to run this on your machine:
Install Maven on your machine.
Download this code repository.
cd <Directory of code repository>
mvn test

That's it!
